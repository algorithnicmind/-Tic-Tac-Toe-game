# Contributing to Tactix Pro 🚀

First off, thank you for considering contributing to **Tactix Pro**! It's people like you that make the open-source community such a fantastic place.

## How Can I Contribute?

### Reporting Bugs
If you find a bug, please open an issue and include:
- A clear title and description.
- Steps to reproduce the bug.
- Any relevant screenshots or error logs.

### Suggesting Enhancements
Have a great idea? Open an issue with the tag `enhancement` and describe:
- The problem you want to solve.
- Your proposed solution.

### Pull Requests (PRs)
1. **Fork** the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the UI remains consistent with the **Glassmorphism** aesthetic.
4. Issue that PR!

## Style Guide
- Use semantic HTML tags.
- Follow the existing Vanilla CSS variable patterns.
- Keep JavaScript functions focused and well-documented.

## Community
Join us in making **Tactix Pro** the best strategy game on the web!

Happy coding! 💻
