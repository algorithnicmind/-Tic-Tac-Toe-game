# Premium Tic-Tac-Toe

A modern, high-end Tic-Tac-Toe game built with Vanilla JS, CSS, and HTML.

## Features
- **Glassmorphism UI**: Beautiful, modern design with blur effects.
- **Score Tracking**: Persistent scores using LocalStorage.
- **Animations**: Smooth pop-in animations for markers and winning highlights.
- **Responsive**: Works on mobile, tablet, and desktop.

## How to Play
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Enjoy the game!
